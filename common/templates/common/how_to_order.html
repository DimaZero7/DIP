Как оформить заказ

    Для того чтобы оформить заказ нужно:
  • Перейти в каталог;
  • Выбрать желаемую категорию;
  • Перейти на детальную страницу товара;
  • Выбрать желаемое количество товара;
  • Добавить товар в корзину;
  • Перейти на страницу корзины;
  • Проверить данные товаров в корзине;
  • Заполнить данные для доставки;
  • По необходимости написать комментарий к заказу;
  • Нажать кнопку заказа товара;
  • Проверить заказ на странице всех заказов;
  • Оплатить заказ;
  • Подтвердить получение заказа.
